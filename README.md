# RankTarget

[**中文**](#中文) | [**English**](#english)

---

## 中文

一个 Muse Dash 的 MelonLoader 模组，用于在游戏结算或游玩时展示当前分数与排行榜目标的差距等信息。

### 功能

- 支持在游戏内展示当前分数与排行榜目标的分差。
- 支持通过 `UserData/RankTarget.cfg` 自由配置字体大小、颜色、名称、样式、描边等。

---

## English

A Muse Dash mod powered by MelonLoader that displays the difference between your current score and the leaderboard target during gameplay or on the results screen.

### Features

- Displays the real-time score gap relative to the target rank on the HUD.
- Fully customizable font size, color, name, style, and outline settings via `UserData/RankTarget.cfg`.
